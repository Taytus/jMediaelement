Embedding a Media Element

HTML: Embed Code

<video src="video.m4v"></video> <video poster="poster-frame.jpg" controls="controls"> <source src="video.ogg"></source> <source src="video.m4v"></source> </video> <video poster="poster-frame.jpg" controls="controls"> <source src="ogg-video-but-has-some-strange-extension.foo" type="video/ogg"></source> <source src="mp4-video-but-has-some-strange-extension.bar" type="video/mp4"></source> </video> <video autoplay="autoplay" src="video.flv"></video>

    • video

  • Attributes:

Javscript

Simply include all mm.*-embed.js files. To use embedding cross-browser compilant fallback, if needed and call the mediaElementEmbed-method on all Mediaelements

$('audio, video').mediaElementEmbed();

mediaElementEmbed:Options

removeControls (boolean: false)
removes the controls-Attribute. For unobtrusive, stylable Controlelements
debug (boolean: false)
Always load the (first) implementet fallback.

jwPlayer extends the mediaElementEmbed:Options with additional options in the jwPlayer-namespace

path (string: 'player.swf')
path to the JW Player swf
vars (object: empty)
additional flashvars used by swfobject to embed JW Player
attrs (object: empty)
additional attrs used by swfobject to embed JW Player
params (object: empty)
additional params used by swfobject to embed JW Player
$('video').mediaElementEmbed({ removeControls: true, jwPlayer: { path: '/myPathToJwplayer/player.swf', params: { wmode: true } } });
